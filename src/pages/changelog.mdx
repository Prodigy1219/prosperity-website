## Pre-Launch

### 2026-05-12 — EvenMoreFish, RoseStacker, Fisher skill expansion

**Fishing**
- **Added**: EvenMoreFish plugin — custom fish with rarity tiers (Common / Rare / Epic / Legendary), biome-specific catches, and direct Vault payouts
- **Custom fish are resource-world only.** Main world fishing returns normal vanilla fish (a designated `fishing_pier` region will open later in the main world for variety)
- **Database** enabled for fish stats and `/emf top` leaderboards
- **Competitions** disabled at launch — will be enabled once player count justifies them
- **Jobs Reborn** Fisher wages still paid on every catch (EMF and Jobs both fire on the same event)

**Fisher Skill Tree expanded with EMF integration**
- **L5 Steady Cast** — +5% Fisher Jobs wage on every catch (new)
- **L35 Dockhand** — +10% Fisher Jobs wage on every catch (stacks with Steady Cast) (new)
- **L40 Deepwater Gear** — 15% chance to upgrade a Common catch to Rare in ocean biomes (new)
- **L45 Treasure Rights** — 10% chance to upgrade a Common/Rare catch to Epic (new)
- **L50 Legendary Angler** — 5% chance to upgrade any catch to Legendary (new)
- L10/L25/L30 (Fish Packer / Bait Saver / Cooler Box) remain placeholders for future systems

**Performance**
- **Added**: RoseStacker — dropped items now merge automatically to reduce entity load. Configured items-only (no mob or block stacking). Jobs Reborn payouts unaffected.

**Other**
- Saddle removed from the admin shop's material buyback list (was prone to villager-rolling exploits). Players can still trade saddles via player ChestShops.
- Architecture document published internally — full plugin stack + interaction map.

### 2026-05-08 — Skill Trees Live, Vehicles, World System & Site Cleanup

**Skill Trees**
- **Live**: Skill tree system is fully implemented — open with `/skills`. 70 nodes across 7 categories (10 per job + 10 Global).
- **Added**: Active skill perks — Headlamp I/II, Portable Smelting, Ore Sorter (Miner); Green Thumb, Crop Packer (Farmer); Steady Hand, Block Recovery, Restoration Expert, Builder Cane (Builder); Replanter, Tree Feller, Axe Care (Woodcutter); Tracker's Eye, Field Dressing, Trail Rations, Arrow Recovery, Beast Lore (Hunter); Sea Legs, Rod Care (Fisher); Sure Footing (Global)
- **Added**: Ore Sorter — named blast furnace. Right-click holding ingots or gems to bulk-compress all into storage blocks.
- **Added**: Crop Packer — named smoker. Same mechanic for crops and farm goods.
- **Added**: Builder Cane — named end rod held in offhand grants +2 block reach.

**Vehicles & Drills**
- **Added**: Mining drills now available in-game — D1 ($1,200), D2 ($2,500), D3 ($5,000) via `/drillShop`. No license required.
- **Added**: Farming tractors — Yellow ($800), Red ($1,800), Green ($3,200) via `/tractorShop`. No license required.
- **Added**: Drill Readiness skill nodes (Miner tree) — unlock purchase eligibility for each drill tier.
- **Rule**: Drills only permitted in the resource world. Tractors only on approved agricultural plots.

**World System**
- **Added**: Resource world and resource Nether — separate worlds for mining and bulk extraction
- **Finalized**: World borders — Main world 4,000 radius, Resource world 2,500 radius, Nether resource 1,250 radius
- **Added**: Main world is government-owned and WorldGuard-protected. Players buy or rent plots through AreaShop signs. Build rights are scoped to your plot only.
- **Added**: The End is locked at launch. Opens Week 2+ via server event.
- **Added**: Resource world resets roughly every 6 weeks. Nether resource world every 8 weeks.

**Website**
- **Updated**: Skill Trees page — all 6 jobs now show actual live skill nodes instead of placeholder license chains
- **Updated**: Getting Started page — fixed job slot count (3 free), added server features table, removed outdated license callout
- **Updated**: Economy page — removed Jobs vs Licenses section (licenses not in Season 0), updated progression model
- **Updated**: Laws page — removed license fine entry, updated Extreme Offenses and penalty escalation
- **Updated**: Contracts page — removed license requirement from Tier 2 description

### 2026-05-07 — Economy System, Laws, Quests & Enchants

**Economy & Jobs**
- **Updated**: Jobs Reborn leveling equation flattened to linear progression across all 6 jobs — early levels easier, curve consistent
- **Updated**: Miner XP rates increased ×1.4 — mining now levels at the intended pace
- **Updated**: Miner level-up rewards wired — titles and profession access unlock automatically at key levels (L10, L15, L20, L21, L30, L35, L40, L45, L50)
- **Added**: Hunter kill payouts reduced ~60%, Fisher payouts reduced ~40% — both now on target with income model

**Enchants & Items**
- **Added**: Soulbound enchantment — admin-only, applied to event rewards, prestige gear, and special tools. Items with Soulbound are kept on death.
- **Added**: Mending blocked server-wide — repair progression is intentional, Mending shortcuts it

**Content**
- **Added**: Starter quest chain — 6 quests covering all four income sources, rewards iron tools, job kit, city stipends, and first week's rent
- **Added**: Public contracts board — 13 launch contracts across all 6 jobs with stated payouts and deadlines

**Website**
- **Added**: Laws page — full fine schedule, restitution system, jail, license suspension, appeal process
- **Updated**: Economy page — 4-source income model, Three-Price Rule, Jobs vs Licenses breakdown
- **Updated**: Skill Trees page — all 6 jobs with correct titles, level-by-level unlocks, and license chains
- **Updated**: Getting Started page — correct job list, `/jobs` commands, income model overview
- **Fixed**: Jobs previously listed as Merchant / Industrialist / Civil Service — corrected to the actual six: Miner, Farmer, Builder, Woodcutter, Hunter, Fisher
