## Pre-Launch

### 2026-05-14 — Plot Plugin Swap: AreaShop → AdvancedRegionMarket

The plot purchase/rent system was migrated from AreaShop 2.6.0 (last released 2019) to **AdvancedRegionMarket (ARM)**, an actively-maintained alternative. AreaShop's age was causing a `BlockPhysicsEvent` NullPointerException on Paper 1.21.8 that was interfering with ChestShop sign placement. ARM uses modern Paper event handlers and integrates the same way (WorldGuard region-backed signs, automatic membership transfer on purchase).

No player-visible change — same buy/rent flow, same per-plot ownership model. Sign tags are slightly different (`[buy]` / `[rent]` instead of `[asbuy]` / `[asrent]`).

---

### 2026-05-13 — Drill and Tractor World Rules Updated

**Drills** are now permitted in the **resource world, nether, and end**. Still not allowed in the main world.

**Tractors** are now permitted in **all worlds**. In the main world, they only work on land you own (your bought or rented plots) because the city's protection rules block block-break elsewhere. Outside the main world, they work freely.

This replaces the previous "tractors only on agricultural plots" rule — agricultural plots still exist as a designated district, but tractor use is no longer restricted to them.

### 2026-05-13 — Referral Rates Raised, 30-Day Launch Bonus Added

Inviter standard reward raised from $300 to **$1,000**. Invitee bonus raised from $150 to **$500**. Milestone bonuses scaled up accordingly.

For the first 30 days post-launch (May 22 – June 21), the inviter side is **doubled** — $2,000 per friend you bring during launch month. Get your friends in early.

### 2026-05-12 — Server Renamed to ProsperitySMP

The server has been renamed from **New Prosperity** to **ProsperitySMP**. Cleaner name, easier to brand, fits Minecraft community naming conventions. All future references will use the new name. (Historical changelog entries have also been updated for consistency.)

**Builder job now pitched directly to skilled builders.** Three pages updated:

### 2026-05-12 — EvenMoreFish, RoseStacker, Fisher skill expansion

**Fishing**
- **Added**: EvenMoreFish plugin — custom fish with rarity tiers (Common / Rare / Epic / Legendary), biome-specific catches, and direct Vault payouts
- **Custom fish are resource-world only.** Main world fishing returns normal vanilla fish (a designated `fishing_pier` region will open later in the main world for variety)
- **Database** enabled for fish stats and `/emf top` leaderboards
- **Competitions** disabled at launch — will be enabled once player count justifies them
- **Jobs Reborn** Fisher wages still paid on every catch (EMF and Jobs both fire on the same event)

**Fisher Skill Tree expanded with EMF integration**
- **L5 Steady Cast** — +5% Fisher Jobs wage on every catch (new)
- **L35 Dockhand** — +10% Fisher Jobs wage on every catch (stacks with Steady Cast) (new)
- **L40 Deepwater Gear** — 15% chance to upgrade a Common catch to Rare in ocean biomes (new)
- **L45 Treasure Rights** — 10% chance to upgrade a Common/Rare catch to Epic (new)
- **L50 Legendary Angler** — 5% chance to upgrade any catch to Legendary (new)
- L10/L25/L30 (Fish Packer / Bait Saver / Cooler Box) remain placeholders for future systems

**Performance**
- **Added**: RoseStacker — dropped items now merge automatically to reduce entity load. Configured items-only (no mob or block stacking). Jobs Reborn payouts unaffected.

**Other**
- Saddle removed from the admin shop's material buyback list (was prone to villager-rolling exploits). Players can still trade saddles via player ChestShops.
- Architecture document published internally — full plugin stack + interaction map.
